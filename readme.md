# editor

[![Go Reference](https://pkg.go.dev/badge/github.com/UltiRequiem/editor.svg)](https://pkg.go.dev/github.com/UltiRequiem/editor)

📖 Read from $EDITOR

For example 👇

```sh
EDITOR=vim go run example/main.go
```

Or...

```sh
EDITOR='code -w' go run example/main.go
```

> The default is "vim"
